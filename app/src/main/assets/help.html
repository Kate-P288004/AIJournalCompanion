AI Journal Companion

Help guide for using the app, sorting, searching, and the emotion chart.

What this app does

Write short journal entries, send them to the AI backend for emotion analysis, and store the results in your history. You can then sort or search your history using different algorithms and view an emotion distribution chart.

How to use it

  • 1
    Write an entry
    Type a journal entry in the text box.
  • 2
    Analyze
    Tap Analyze to get an emotion label and short advice from the AI backend.
  • 3
    Sort history
    Open Tools, choose a sorting method, then tap Sort History by Emotion.
  • 4
    Search history
    Type an emotion (example: JOY) and choose a search structure, then tap Search.
  • 5
    View chart
    Tap the chart icon to see the distribution of emotions and the percentage list.
Tip: emotion matching is not case-sensitive, so joy, JOY, and Joy all work the same.

Algorithms in this project

These algorithms are implemented in code and used inside the app:

Bubble Sort Insertion Sort Selection Sort Binary Tree Search HashMap Search Doubly Linked List Search
