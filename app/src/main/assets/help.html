AI Journal Companion

About

AI Journal Companion allows you to write journal entries, analyze emotional tone using AI, and organize your entries using sorting and searching algorithms.

How to Use

  • Write a journal entry and press Analyze.
  • Use Sort to organize entries by emotion.
  • Use Search to find entries by emotion using different algorithms.
  • Tap Show Chart to view emotion distribution.

Algorithms Used

  • Bubble Sort
  • Insertion Sort
  • Selection Sort
  • Binary Tree Search
  • HashMap Search
  • Doubly Linked List Search
